### 🚀 Cara Menggunakan

1. **Clone atau Download Repository**
   ```bash
   git clone https://github.com/munawarjr021-cpu/portfolio-munawar.git
   cd portfolio-munawar
   ```

2. **Buka index.html di Browser**
   - Cukup buka file `index.html` langsung di browser Anda
   - Atau gunakan Live Server extension jika menggunakan VS Code

3. **Edit Konten**
   - Klik pada teks dengan background hijau muda untuk mengedit
   - Konten akan disimpan otomatis di localStorage browser
   - Untuk perubahan permanen, edit file HTML langsung

4. **Upload Foto**
   - Ganti URL placeholder dengan foto nyata
   - Edit di bagian `.image-placeholder`

### 🌐 Deploy ke GitHub Pages

1. Push repository ke GitHub:
   ```bash
   git add .
   git commit -m "Initial commit: Portfolio website"
   git push origin main
   ```

2. Di GitHub repository settings:
   - Pergi ke **Settings > Pages**
   - Source: Main branch
   - Simpan

3. Website akan live di: `https://munawarjr021-cpu.github.io/portfolio-munawar/`

### 🎨 Customization

#### Mengubah Warna
Edit variabel CSS di `css/style.css` bagian `:root`:
```css
:root {
  --primary: #0F766E;
  --primary-light: #14B8A6;
  --secondary: #14B8A6;
  /* ... */
}
```

#### Menambah Section Baru
Gunakan struktur HTML yang sudah ada sebagai template. Ikuti pattern:
```html
<section id="nama" class="nama section-padding">
  <div class="container">
    <h2 class="section-title">Judul Section</h2>
    <!-- Konten di sini -->
  </div>
</section>
```
